DIRECTORIO DE FUENTES - MultiTaller

Este directorio debe contener los siguientes archivos de fuente:

1. FUENTE INTER (Google Fonts)
   Descargar de: https://github.com/rsms/inter/releases
   
   Archivos requeridos:
   - Inter-Regular.woff2    (peso 400)
   - Inter-Medium.woff2     (peso 500)
   - Inter-SemiBold.woff2   (peso 600)
   - Inter-Bold.woff2       (peso 700)

2. BOOTSTRAP ICONS
   Descargar de: https://github.com/twbs/icons/blob/main/font/fonts/bootstrap-icons.woff2
   
   Archivo requerido:
   - bootstrap-icons.woff2

INSTRUCCIONES:
1. Descargar los archivos mencionados
2. Colocarlos en este directorio: /multitaller/static/fonts/
3. Reiniciar la aplicación MultiTaller
4. Limpiar caché del navegador (Ctrl+F5)

NOTA: Sin estas fuentes, el sistema mostrará fuentes alternativas del sistema
y los íconos no se visualizarán correctamente.
